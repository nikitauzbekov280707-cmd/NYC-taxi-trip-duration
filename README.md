# 🚕 NYC Taxi Trip Duration Prediction

End-to-end проект по машинному обучению для предсказания длительности поездки такси в Нью-Йорке.

Проект включает полный ML pipeline: от анализа данных до развёртывания модели через API и веб-приложение.

---

# 📌 Описание проекта

Цель проекта — предсказать длительность поездки такси в Нью-Йорке на основе информации о поездке:

- ⏰ время начала поездки  
- 📍 координаты посадки и высадки  
- 👥 количество пассажиров  
- 🚕 дополнительные метаданные поездки  

Проект основан на соревновании Kaggle **NYC Taxi Trip Duration**.

В рамках проекта реализовано:

- 📊 разведочный анализ данных (EDA)
- ⚙️ feature engineering
- 🤖 обучение модели градиентного бустинга
- 🏆 оценка модели на Kaggle
- 🌐 API для инференса модели
- 💻 веб-интерфейс для пользователя

---

# 📊 Этапы проекта

## 🔎 1. Анализ данных (EDA)

Был проведён анализ датасета:

- исследование распределения длительности поездок
- поиск выбросов
- анализ координат поездок
- анализ временных паттернов

---

## ⚙️ 2. Feature Engineering

Были созданы дополнительные признаки.

### ⏱ Временные признаки

- час начала поездки
- день недели
- месяц
- признак выходного дня

### 📍 Географические признаки

- расстояние между точками (Haversine distance)
- кластеризация координат (KMeans)


---

## 🤖 3. Модель

Основная модель:

**LightGBM Regressor**


---

## 🧪 4. Валидация

Для корректной оценки модели была реализована стратегия валидации, учитывающая временную структуру данных.

Это позволило:

- избежать утечек данных
- получить более реалистичную оценку качества модели
